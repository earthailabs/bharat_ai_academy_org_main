Foundation

Set up your Google account professionally

Your account name, security, and browser habits affect every Google tool.

  1. Use a professional email: firstname.lastname@gmail.com
  2. Add recovery phone and email so you can recover the account.
  3. Turn on 2-Step Verification at myaccount.google.com/security.
  4. Use Chrome profiles to keep personal, study, and work accounts separate.
  5. Open the 9-dot launcher to access all Google apps.
Avoid: sharing passwords, using unclear email names, or staying signed in on public computers.

Gmail practice

Inbox zero without deleting everything

  • Archive: remove from inbox but keep searchable.
  • Label: categorize mail—Classes, Jobs, Projects.
  • Filter: automatically label or archive matching mail.
  • Search: from:, has:attachment, after:.

Mini activity: Send a professional email

  1. Subject: Class 5 Assignment Submission – Your Name
  2. Write a greeting and one-sentence purpose.
  3. Add a Drive link—not a huge attachment.
  4. Request confirmation and add a closing.
  5. Schedule it for a suitable time.

Google Docs

Write, collaborate and publish

Use Docs for resumes, reports, meeting notes, proposals, assignments and SOPs.

Best feature: Multiple people can edit the same file without sending different copies.
  • Styles: Title, Heading 1 and Heading 2 create structure.
  • Comments: tag a teammate with +email and assign an action.
  • Suggesting mode: propose edits without replacing original text.
  • Version history: see who changed what and restore older versions.
  • Smart chips: insert people, dates, files and dropdowns.
  • Download: export as PDF, DOCX or plain text.

Google Sheets

Turn rows into decisions

Rows

One record per row: one student, sale or expense.

Columns

One field per column: Name, Date, Amount.

Cells

A1 means column A, row 1. A1:C10 is a range.

Data types

Text, number, date, currency, percentage, checkbox.

  • Sort changes order; Filter temporarily hides unmatched rows.
  • Freeze the header row so column names remain visible.
  • Use data validation for clean dropdown choices.
  • Use conditional formatting to highlight late, low or high values.

Sheets practice

Four formulas every beginner should know

=SUM(D2:D20)
Total sales =AVERAGE(D2:D20)
Average sale =COUNTIF(C2:C20,"Paid")
Paid records =IF(D2>=500,"High","Normal")
Classify values

Mini project: Class tracker

Columns: Student, Date, Attendance, Assignment Score, Status. Add dropdowns, average score and conditional formatting.

Charts

Bar = compare categories. Line = trend over time. Pie = limited part-to-whole. Always add a clear title.

AI use: Ask Gemini or ChatGPT to explain a formula, but verify the cell ranges before trusting it.

Google Calendar

If it matters, schedule it

Calendar turns intentions into time blocks and reduces missed classes, meetings and deadlines.

  1. Create an event with clear title, date and time.
  2. Add guests, location, notes and a Drive file.
  3. Add a Google Meet link for online meetings.
  4. Set notifications: one day before + 15 minutes before.
  5. Use recurring events for weekly classes or habits.
  6. Create separate calendars for Personal, Study and Work.
Use case: Schedule “LinkedIn post” after every class—repeat weekly.

YouTube channel setup

Create your public learning channel

A channel can become your video portfolio, teaching platform and professional identity.

  1. Sign in to YouTube with the correct Google account.
  2. Profile picture → Create a channel.
  3. Choose professional channel name and handle.
  4. Add profile photo, banner and a clear description.
  5. Open YouTube Studio to manage content and settings.
  6. Verify the channel when YouTube requests it for more features.
